# Kyoungbin Park Portfolio

`Next.js` 기반으로 만든 개인 포트폴리오 사이트입니다.

## Tech Stack

- `Next.js`
- `React`
- `TypeScript`
- `Tailwind CSS`
- `motion`
- `react-markdown`
- `lucide-react`

## Sections

- Home
- Skills
- Resume
- Portfolio
- Education
- Inspire
- Contact

## Run

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000` 을 열면 됩니다.

타입 체크:

```bash
npm run lint
```

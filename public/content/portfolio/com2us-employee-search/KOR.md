# 컴투스 사내 직원 검색용 Self-Updating RAG

## 프로젝트 개요

이 프로젝트는 특정 업무의 담당자를 더 빠르게 찾을 수 있도록 만든 사내 직원 검색 AI 챗봇입니다. 컴투스 인턴 기간 동안 AI/ML 엔지니어 선임의 멘토링 아래 1인 개발로 진행했고, 최종 결과물은 실제 사내 개발자들이 사용하는 서비스로 운영되었습니다.

## 해결하려던 문제

프로젝트 담당자가 자주 바뀌거나 여러 부서에 걸쳐 협업이 이루어지는 환경에서는 특정 업무의 담당자를 찾는 데 많은 시간이 들었습니다. 이를 줄이기 위해 자연어 질문만으로 가장 관련도 높은 직원을 찾아주는 검색 시스템을 설계했습니다.

## 시스템 구성

![컴투스 결과 화면 01](/content/portfolio/com2us-employee-search/result01.png "w=1080 h=620")
![컴투스 결과 화면 02](/content/portfolio/com2us-employee-search/result02.png "w=1080 h=620")

- 주간 업무 보고서를 Qwen2.5-32B-Instruct-GPTQ-Int4로 요약
- 요약 결과를 JSON 형식으로 정규화
- 정규화된 데이터를 벡터화하여 임베딩 공간에 저장
- 사용자의 질문과 가장 유사한 직원 정보를 검색해 반환

이 구조를 통해 매주 새로운 보고서가 반영되는 self-updating RAG 파이프라인을 구현했습니다.

## 모델 개선

게임 개발 도메인 특성상 유사 용어가 많아 검색 품질을 높이는 작업이 중요했습니다. 이를 위해 multilingual-e5-large 임베딩 모델을 사내 주간보고서 데이터로 파인튜닝했고, 과거 데이터를 학습용으로, 최근 데이터를 평가용으로 활용했습니다.

## 성능 지표

- recall@10: 0.04 -> 0.18
- ndcg@10: 0.02 -> 0.11
- 질문과 실제 담당자 간 cosine similarity: 0.708 -> 0.869

## 결과

최종 시스템은 실제 컴투스 내부에 배포되어 개발자들이 사용하는 사내 AI 서비스로 활용되었고, 사내 AI 서비스 확산에도 기여했습니다.

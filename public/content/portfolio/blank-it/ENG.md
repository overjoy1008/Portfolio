# Blank-It

## Overview

Ttuleoboka is an LLM-based web service that generates fill-in-the-blank exam questions from uploaded university lecture PPT materials.

## Team Role

This project was built by a four-person team consisting of planning, design, and development members. I worked as the Frontend Developer and Prompt Engineer.

## What I Built

![Blank-It quiz generation mockup](https://picsum.photos/seed/blankit-quiz/1200/700)

- Implemented the frontend with React, Tailwind CSS, and Next.js
- Designed prompt behavior for quiz generation using ChatGPT-3.5
- Built a user flow where students could upload lecture materials and receive practice questions based on key concepts

## Backend

- Firebase

## Outcome

- More than 2,500 page views within the first week after launch
- More than 750 visitors
- More than 1,200 generated study questions

## Impact

The service was promoted on bulletin boards across major college buildings at Korea University and was used as a practical study tool for students.

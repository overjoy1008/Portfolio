**기간:** 2024.03.26 - 2024.04.07 (2주)

**소속:** 고려대학교 생성형 AI 학회 팀 프로젝트

**역할:** Frontend Developer, Component Development, Prompt Engineering

## 프로젝트 개요

뚫어보카는 수업 PPT를 업로드하면 핵심 내용을 바탕으로 10개의 빈칸 채우기 문제를 자동 생성해 주는 AI 기반 학습자료 문제 생성 서비스입니다. 고려대학교 생성형 AI 학회 활동 중 4인 팀으로 2주 동안 기획, 개발, 배포를 진행했으며, DDD 원칙을 바탕으로 Next.js, React, Firebase 스택을 사용해 빠르게 출시 가능한 형태로 구현했습니다.

- 수업 PPT를 기반으로 핵심 키워드를 추출하고 빈칸 채우기 문제 10개를 생성하는 학습 플로우를 구현했습니다.
- 어떤 분야의 전공이든 암기 학습에 활용할 수 있도록 범용 학습 보조 서비스로 기획했습니다.
- 4인 팀(개발 2, 디자인 1, 기획 1)으로 짧은 기간 안에 실제 사용 가능한 웹앱 출시를 목표로 개발했습니다.

## Pain Point 및 목표 설정

- **기존 문제점:** 시험 대비나 복습을 위해 강의 자료를 직접 정리하고 문제를 만드는 과정은 시간이 많이 들고, 전공마다 자료 형식이 달라 범용적으로 활용할 수 있는 암기 도구가 부족했습니다.
- **제약 조건:** 2주 안에 기획, 개발, 배포를 마쳐야 했고, 업로드되는 자료가 PDF뿐 아니라 이미지와 텍스트 파일까지 포함되어 다양한 입력 포맷을 안정적으로 처리해야 했습니다.
- **해결 목표:** 다양한 입력 자료를 빠르게 인식해 학생들이 바로 사용할 수 있는 문제 생성 웹앱을 완성하고, 짧은 개발 일정 안에서도 실제 배포까지 이어지는 서비스를 만드는 것이 목표였습니다.

## 개발 구현 내용

- **프론트엔드 및 컴포넌트 구현:** Next.js와 React를 기반으로 전체 사용자 플로우를 구현했고, 버튼을 포함한 주요 UI 컴포넌트를 직접 개발해 빠른 제작 속도와 일관된 사용 경험을 확보했습니다.
- **다양한 입력 포맷 지원:** PDF, 이미지, 텍스트 파일 입력을 받을 수 있도록 구성했고, OCR 기반 텍스트 추출을 통해 카메라 촬영 이미지까지 학습 자료로 활용할 수 있게 했습니다.
- **문제 생성 프롬프트 설계:** 업로드된 강의 자료에서 핵심 키워드를 안정적으로 추출하고 빈칸 문제 퀄리티를 높일 수 있도록 ChatGPT 3.5 Turbo 기반 프롬프트를 설계하고 개선했습니다.
- **사용자 경험 최적화:** 반응형 UI를 적용해 모바일과 데스크톱 환경을 모두 지원했고, 홈 이동, 뒤로가기, 파일 용량 제한 안내 등 실제 사용 흐름에 필요한 UX를 보완했습니다.

## 트러블슈팅 및 해결 과정

- **문제 상황:** PDF를 파싱하는 과정에서 글자가 깨지거나 OCR이 실패하는 경우가 있었고, 원본 PDF의 줄바꿈 구조나 필기체 표현 때문에 텍스트 추출 품질이 흔들렸습니다.
- **원인 분석:** 문제 생성 품질 저하의 핵심 원인은 LLM 자체보다 입력 텍스트 품질에 있었고, 특히 PDF별 구조 차이와 시각적 요소가 파싱 결과를 불안정하게 만든다는 점을 확인했습니다.
- **해결 방안:** 프롬프트 엔지니어링과 전처리 기준을 함께 조정하면서 PDF 파싱 결과를 반복 점검했고, 줄바꿈 텍스트 처리 버그와 대용량 이미지 처리 이슈를 포함해 케이스별로 대응해 문제 생성 결과를 안정화했습니다.
- **결과:** 다양한 입력 포맷에서도 문제 생성 품질을 유지할 수 있었고, 2주 안에 실제 배포 가능한 수준의 학습 서비스를 완성했습니다.

## 주요 성과

- **빠른 출시:** 2024년 1학기 생성형 AI 학회 활동 중 2주 만에 기획, 개발, 배포까지 완료했습니다.
- **초기 반응:** 출시 후 일주일 동안 2,500회 이상의 조회수와 750명 이상의 방문자를 기록했습니다.
- **사용 지표:** 1,200개 이상의 학습 문제가 생성되며 실제 학습 보조 도구로 활용되었습니다.
- **홍보 및 확산:** 고려대학교 주요 단과대 학생회관 게시판 홍보를 통해 실제 학생 사용자 유입으로 이어졌습니다.
